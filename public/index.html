Spongebob Quotes API Instructions

These are all GET requests that return JSON

To get quotes use url: https://spongebob-quotes.onrender.com//api/quotes

To get random quote use url: https://spongebob-quotes.onrender.com/api/quotes/random

To get quote by name use url where query parameter name=character will be a name of a character: https://spongebob-quotes.onrender.com/api/quotes/search/?name=spongebob
